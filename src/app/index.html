A DISTRIBUTED TRASACTIONAL DATABASE

TiDB is an open-source NewSQL distributed database solution with a full support for
crossdatacenter replication and dsaster recovery. Inspired by Google F1/Spanner,
TiDB features in horizontal scalability, consistent distributed transactions,
and synchronous replication of multiple replicas.
